# Digital Card – Linktree Style

Una **tarjeta digital interactiva** diseñada para centralizar enlaces importantes en un solo lugar, similar a plataformas como Linktree.

Este proyecto funciona como una **página de perfil simple y elegante** donde los usuarios pueden acceder rápidamente a redes sociales, portafolios, tiendas o cualquier enlace relevante desde un solo link.

---

## ✨ Características

* Diseño limpio y enfocado en conversión
* Estructura ligera en **HTML, CSS y JavaScript**
* Metodología **BEM** para organización del CSS
* Fácil de personalizar para cualquier marca o persona
* Ideal para usar como **bio link en redes sociales**

## 🎯 Objetivo del proyecto

El objetivo de esta web es servir como **digital card personal o de marca**, permitiendo compartir múltiples enlaces desde una sola página optimizada para dispositivos móviles.

Este tipo de página es ideal para:

* Creadores de contenido
* Emprendedores
* Marcas personales
* Tiendas online
* Freelancers

---

## 🛠 Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Metodología BEM

---

## 📱 Casos de uso

Puede utilizarse como:

* Link en biografías de redes sociales
* Tarjeta digital de contacto
* Página de enlaces para portafolio
* Página de acceso rápido a productos o servicios
